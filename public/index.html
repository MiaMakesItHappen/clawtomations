Need instructions?

App docs live in README.md inside the project folder. Obsidian tracking note is Projects/clawtomations.md.

  • Run once: node src/cli.js run --workflow workflows/sample-workflow.yaml
  • Start browser session capture: node src/cli.js capture --site &lt;alias&gt; --url &lt;start-url&gt;
  • Open button UI: node src/cli.js serve then open http://localhost:8787
  • Schedule daily run: node src/cli.js schedule --workflow workflows/sample-workflow.yaml --time 09:00

Advanced / troubleshooting

OpenClaw adapter: Set openclaw.command in the workflow or CLAWTOMATIONS_OPENCLAW_COMMAND to force a custom invocation.

  • OpenClaw v1 style sample: openclaw run --workflow {{workflow.path}} --output {{workflow.output}}
  • OpenClaw v2 style sample: openclaw agent run --workflow {{workflow.path}} --output {{workflow.output}}
  • Build + send with local command: node src/cli.js openclaw --workflow workflows/sample-workflow.yaml

Troubleshooting:

  • Run fails at login: re-run capture and update auth/*.json.
  • Session errors: delete old auth state and recapture for that site.
  • Browser startup fail: confirm npm install and Playwright install completed.
  • Missing site path: verify each workflow uses a valid authState path.

Example workflow snippet:

settings:
  headless: true
  outputDir: "outputs"

sites:
  - name: example
    authState: "auth/example-session.json"
    requiresLogin: true
    steps:
      - action: navigate
        url: "https://example.com"
      - action: copy
        selector: "h1"
        key: "heading"
      - action: screenshot
        file: "home.png"
        key: "homepage"
